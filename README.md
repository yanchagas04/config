#  Guia de Configuração ⚙️

Esse guia serve para eu ter acesso fácil aos comandos que precisarei executar caso precise reinstalar o sistema novamente.

### Como funciona?

Esse repositório está sendo construído utilizando uma branch pra cada SO que eu quiser colocar arquivos de configuração, sendo a branch `main` responsável pelas configurações universais (servindo para todos os SOs)
